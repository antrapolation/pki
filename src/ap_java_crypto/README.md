# ApJavaCrypto

ApJavaCrypto is the module that invoke Java Cryptographic Library for elixir
In term of completeness of support cryptographic algorithm, Java BouncyCastle still at the forefront of the curve

Java also the platform implementing the KAZ algorithm, including X.509.
