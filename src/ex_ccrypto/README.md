# ExCcrypto

ExCcrypto is the high level cryptographic API for Elixir project.

This has taken shape after the proven benefit of Java Cryptographic Extension (JCE) design that other user of the cryptographic algorithm may not need to be so technical instead a high level and standardize cryptographic API would be very helpful in building security solution on top of this layer.
