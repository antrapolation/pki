# StrapProcReg

Security TRansformation Platform (STRAP) - Process Registry

Handle the process registry of Elixir/Erlang processes.

Inspired by Sasa Juric talk that this is controlled by developer and should try to write this registry.

However stress test not being done as of Jun 2025.
